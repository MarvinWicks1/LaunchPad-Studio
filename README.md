# LaunchPad Studio

This is a fictional landing page I built to practise creating a more complete business-style website.

The idea was to make something that could work for a small digital studio or startup service. I focused on making the page feel more polished than a basic layout, with proper sections, pricing cards, testimonials, FAQs, and a strong call to action.

## What I practised

- Building a full landing page layout
- Creating reusable card sections
- Making the design responsive
- Using only HTML and CSS
- Improving spacing, hierarchy, and calls to action

## Tech used

- HTML
- CSS
- GitHub Pages

## What I would improve next

I’d like to add a working contact form or some small JavaScript interactions, such as animated FAQs or a mobile menu.
